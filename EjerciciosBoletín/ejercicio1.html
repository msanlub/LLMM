HTML

HTML son las siglas de HyperText Markup Language, que puede traducirse como lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML se emplea para las siguientes páginas web. Es muy fácil ver el código HTML de una página web, la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mismo navegador,pero suelen tener un nombre similiar.

HTML se compone de etiquets que se escriben entre los símbolos menor que y mayor que.

Historia de HTML

Los inicios del lengaje HTML se remontan al año 1990, cuando Tim Bernes-Lee creó la primer página web.

Versiones de HTML

Tim Bernes-Lee definió la primera versión de HTML en el año 1991.
En la actualidad, la última versión de HTML es HTML5.
